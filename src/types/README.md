# Types

This directory contains TypeScript type definitions and interfaces for the proxy service.

## Type Files

- **requests.ts** - HTTP request/response types
- **cache.ts** - Cache entry and statistics types
- **context.ts** - Code context and normalization types
- **copilot.ts** - GitHub Copilot API types
- **config.ts** - Configuration structure types
- **metrics.ts** - Metrics and aggregation types
- **health.ts** - Health monitoring and diagnostic types

All component interfaces are defined here to ensure clear contracts between components and enable easy testing.
